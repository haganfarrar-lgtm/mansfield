# Mansfield Landscaping Website

Professional landscaping website for Mansfield Landscaping, serving Mansfield, TX and surrounding DFW areas.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Deployment:** Vercel

## Features

- 🏠 Dynamic city pages for 50+ service areas
- 🛠️ 30+ service pages with photo galleries
- 📝 Blog with 20+ SEO-optimized articles
- 📱 Fully responsive design
- 🔍 Comprehensive SEO optimization
- 📊 Structured data (Schema.org)
- 🗺️ Auto-generated sitemap
- ⚡ Optimized performance

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

### Environment Variables

No environment variables required for basic setup.

## Deployment

This project is configured for deployment on Vercel:

1. Push to GitHub
2. Connect repository to Vercel
3. Deploy automatically

## Project Structure

```
├── app/                    # Next.js App Router pages
│   ├── blog/              # Blog pages
│   ├── cities/            # City and service pages
│   └── page.tsx           # Homepage
├── components/            # React components
├── data/                  # Static data (cities, services, blog posts)
├── lib/                   # Utility functions and SEO helpers
├── public/               # Static assets
│   └── images/           # Service and crew images
└── vercel.json           # Vercel configuration
```
